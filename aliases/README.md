Optional **abbreviation** aliases: **SO** → SOMOPEN, **SS** → SOMSAVE, **SI** → SOMINSERT.

Import `SOM-Rhino-Aliases.txt` in Rhino via **Tools → Options → Aliases → Import**.

**Note:** Rhino does not allow alias names that match built-in command names (e.g. you cannot alias "Open" to SOMOPEN). The single entry point from the UI is provided by the plugin’s **menu items** (File → SOM Open, etc.), not by aliases.

# Source

Place Rhino C++ plugin source files here:

- `SOMRhinoToolsApp.h` / `.cpp` – Application class
- `SOMRhinoToolsPlugIn.h` / `.cpp` – Plugin class
- `cmdSOMOpen.cpp` – SOMOPEN command
- `cmdSOMSave.cpp` – SOMSAVE command
- `cmdSOMInsert.cpp` – SOMINSERT command
- `SomHelpers.h` / `.cpp` – Shared helpers (IsGoogleDrivePath, RhlExists, etc.)

Create the initial project using the **Rhino 3D Plugin (C++)** template in Visual Studio, then add/modify commands as documented in Architecture.md.
